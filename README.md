# 陆战兵棋SDK

## 项目结构

`ai/`：提供了一个可以参考的AI样例，开发者需在此文件夹中实现自己的AI算法 

`core/`：开发平台提供的核心模块，主要存放平台提供的服务

`docs/`：相关文档

`requirements.txt`：SDK运行所需的环境依赖

`run_offline_games.py`：单机启动引擎，AI本地训练的入口程序

## 运行方式

在`Linux`系统下，默认启动两个`DemoAgent`进行对抗

```bash
python run_offline_games.py
```
